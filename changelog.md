# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] — 2026-03-11

### Added
- Initial release
- Connect to Tautulli via URL and API key
- User selection from Tautulli instance
- Optional date range filtering (from / to)
- CSV export with `Title`, `Year`, `WatchedDate` columns
- Automatic same-day duplicate removal
- Optional re-watch deduplication (keep first watch only)
- Direct link to Letterboxd import page after export
- Inline error messages with troubleshooting tips
- Fully client-side — no server or external dependencies
